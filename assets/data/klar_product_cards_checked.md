# Картки товарів KLAR EcoSensitive

Дані звірено з офіційними картками виробника AlmaWin / KLAR EcoSensitive. Зовнішній вигляд паковання та окремі характеристики партії можуть відрізнятися, тому перед використанням завжди перевіряйте етикетку конкретного засобу.

---

## 1. KLAR Universal Waschmittel Waschnuss, 0,75 л — універсальна рідина для прання

**Склад**

- **5–15%** аніонні ПАР: екстракт мильного горіха та рослинні сульфати жирних спиртів.
- **<5%** неіоногенні ПАР: цукрові ПАР.
- Мило з рослинних олій*.
- Вода, натрієві солі лимонної кислоти, етанол, лимонна кислота, ксантанова камедь, молочна кислота.
- **100% інгредієнтів мають природне походження.**

\* З контрольованого органічного сільського господарства.

**Інструкція із застосування**

- Налийте засіб у відсік пральної машини. Для ручного прання розчиніть його у воді.
- Для завантаження **4–5 кг**, м’яка вода: легке/середнє забруднення — 30 мл, сильне — 45 мл.
- Середня жорсткість: легке — 30 мл, середнє — 45 мл, сильне — 60 мл.
- Жорстка вода: легке — 45 мл, середнє — 60 мл, сильне — 75 мл.
- Для завантаження 2–3 кг зменште дозу на 15 мл; для 6–8 кг додайте 45 мл.
- Для ручного прання використовуйте 10 мл засобу на 5 л води.

**Рекомендації та обмеження**

- Підходить для білих, чорних і кольорових тканин із натуральних, змішаних та синтетичних волокон за температури 20–95 °C.
- Не використовуйте для вовни та шовку. Дотримуйтеся рекомендацій на ярликах одягу.
- Для білої білизни виробник радить додавати KLAR Fleckensalz.
- Засіб спричиняє серйозне подразнення очей. Зберігайте в недоступному для дітей місці. У разі потрапляння в очі промивайте водою кілька хвилин.

**Склад**

- **5–15%** мило та мило з рослинних олій*.
- **<5%** аніонні ПАР: рослинні сульфати жирних спиртів та екстракт мильного горіха.
- **<5%** неіоногенні ПАР: рослинні цукрові ПАР.

- **100% інгредієнтів мають природне походження.** Оптичні відбілювачі та ензими виробником не зазначені.

\* З контрольованого органічного сільського господарства.

**Інструкція із застосування**

- Для завантаження **4–5 кг**, м’яка вода: легке забруднення — 40 мл, середнє — 50 мл, сильне — 75 мл.
- Середня жорсткість: легке — 50 мл, середнє — 60 мл, сильне — 90 мл.
- Жорстка вода: легке — 60 мл, середнє — 70 мл, сильне — 100 мл.

- Зберігайте порошок у сухому місці та бережіть від вологи.
- Для вираженого ефекту білизни оберіть засіб із відбілювальним компонентом: цей порошок призначений насамперед для кольорових тканин.

**Інструкція із застосування**

- За сильного забруднення або наявності сечового каменю залиште засіб на ніч, потім почистьте й змийте.

**Рекомендації та обмеження**

- Використовуйте лише на кислотостійких поверхнях.
